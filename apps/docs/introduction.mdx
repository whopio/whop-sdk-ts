Why build on **Whop**?

1. **Authentication**: Zero authentication required. Instantly pull a load of user information.
2. **Payments**: Accept payments and make payouts with extremely low effort.
3. **Distribution**: Reach unheard of distribution channels for your app, tool, game, or agent.


Get started below:

<CodeGroup>

```bash Next.js
npx create-next-app@latest whop-app -e https://github.com/whopio/whop-nextjs-app-template
```

</CodeGroup>


<CodeGroup>

```bash pnpm
pnpm i
```

```bash npm
npm i
```

```bash yarn
yarn i
```

</CodeGroup>

<CodeGroup>

```bash pnpm
pnpm dev
```

```bash npm
npm run dev
```

```bash yarn
yarn dev
```

</CodeGroup>

Now open http://localhost:3000 and follow the directions.
