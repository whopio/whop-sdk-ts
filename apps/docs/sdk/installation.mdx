description: "Getting started developing an App on Whop"
---

We recommend you use our TS SDK to build your app. It's a wrapper around the raw API that makes it easier to use.

<CodeGroup>

```bash pnpm
pnpm install @whop/api && pnpm install -D @whop-apps/dev-proxy
```

```bash npm
npm install @whop/api && npm install -D @whop-apps/dev-proxy
```

```bash yarn
yarn add @whop/api && yarn add -D @whop-apps/dev-proxy
```

</CodeGroup>
