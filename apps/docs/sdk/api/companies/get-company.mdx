---
title: Get Company
description: Fetches a company based on the ID
---

```typescript
import { whopSdk } from "@/lib/whop-sdk";

const result = await whopSdk.companies.getCompany({
	// The ID of the company to fetch.
	companyId: "biz_XXXXXXXX" /* Required! */,
});

```
