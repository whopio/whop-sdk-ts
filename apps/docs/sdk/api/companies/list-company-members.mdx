---
title: List Company Members
description: Fetches a company based on the ID
---
<Note>This operation is only available on the server.</Note>
```typescript
import { whopSdk } from "@/lib/whop-sdk";

const result = await whopSdk.companies.listCompanyMembers({
	// The ID of the company to fetch.
	companyId: "biz_XXXXXXXX" /* Required! */,

	first: 10,

	last: 10,

	after: "pageInfo.endCursor",

	before: "pageInfo.startCursor",

	searchQuery: "some string",
});

```
