---
title: List User Socials
description: Fetch a specific user.
noindex: true
---
<Note>This operation is only available on the server.</Note>
### Usage

```typescript
import { whopSdk } from "@/lib/whop-sdk";

const result = await whopSdk.users.listUserSocials({
	// ID of the user by tag or the username.
	userId: "user_XXXXXXXX" /* Required! */,

	after: "pageInfo.endCursor",

	before: "pageInfo.startCursor",

	first: 10,

	last: 10,
});

```

### Example output
